Execute https://wren.io/[Wren Scripts] within your https://livebook.dev/[Livebook].

## Installation

1. Install https://livebook.dev/[Livebook].
2. Open `wren.livemd`.
3. Choose your wren binary.
4. Run the cells and enjoy!.
